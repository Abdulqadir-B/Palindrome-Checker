## 💡 What’s a Palindrome?

A palindrome is a word, phrase, number, or sequence that reads the same backward as forward (ignoring spaces, punctuation, and capitalization).

**Examples:**
- `racecar`
- `A man a plan a canal Panama`
- `12321`
